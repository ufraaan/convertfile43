# File Converter — macOS

A macOS-native file conversion app with Finder right-click integration. Convert audio, video, images, PDFs, and Office documents — directly from the Finder context menu.

Inspired by [FileConverter](https://github.com/Tichau/FileConverter) for Windows.

## Requirements

- **macOS 14+ (Sonoma)**
- **Xcode 16+** (download from [Mac App Store](https://apps.apple.com/app/xcode/id497799835) or [Apple Developer](https://developer.apple.com/xcode/))
- **XcodeGen** – `brew install xcodegen`

## Quick Start

```bash
# 1. Generate Xcode project
xcodegen generate

# 2. Download middleware binaries (ffmpeg, imagemagick, ghostscript)
./Middleware/Scripts/download-binaries.sh

# 3. Open in Xcode
open FileConverter.xcodeproj

# 4. Select scheme "FileConverter" and run (⌘R)
```

## First Run

1. Launch FileConverter.app once
2. Go to **System Settings → Privacy & Security → Extensions → Finder Extensions**
3. Enable **"File Converter Finder Sync"**
4. Right-click any file in Finder → **Convert with File Converter** → pick a preset

## Project Structure

```
FileConverter/
├── FileConverter.xcodeproj        # Generated by XcodeGen
├── project.yml                    # XcodeGen manifest
├── FileConverter/                 # Main app (SwiftUI)
│   ├── Models/                    # OutputType, ConversionPreset, ConversionJob, etc.
│   ├── Services/                  # FFmpegService, ImageMagickService, GhostscriptService, etc.
│   ├── ViewModels/                # MainViewModel, SettingsViewModel, etc.
│   ├── Views/                     # SwiftUI views
│   └── Utilities/                 # ProcessRunner, BundlePaths
├── FinderSyncExtension/           # Finder Sync Extension (right-click menu)
├── Middleware/
│   ├── binaries/                  # Bundled ffmpeg, magick, gs
│   └── Scripts/download-binaries.sh
└── Installer/create-dmg.sh
```

## Architecture

```
User right-clicks in Finder
  └─► Finder Sync Extension reads presets from App Group cache
       └─► Shows "Convert with File Converter ▸" submenu
            └─► Writes request JSON to App Group shared container
                 └─► Main app picks up request
                      └─► Spawns ffmpeg/magick/gs process
                           └─► Shows progress in SwiftUI
                                └─► macOS notification on completion
```

## Conversion Capabilities

| Category | Input | Output | Engine |
|---|---|---|---|
| Audio | mp3, wav, flac, ogg, m4a, wma, aac | MP3, AAC, FLAC, OGG, WAV | ffmpeg |
| Video | mp4, mkv, avi, mov, webm, ogv, wmv, flv | MP4, MKV, AVI, WebM, OGV | ffmpeg |
| Image | jpg, png, webp, avif, bmp, tiff, gif | JPG, PNG, WebP, AVIF, ICO, GIF | ImageMagick |
| Document | pdf, jpg, png | PDF ⇄ PNG/JPG | ImageMagick/Ghostscript |
| Office* | doc, docx, odt, ppt, pptx, odp, xls, xlsx, ods | PDF | LibreOffice |

\* Requires LibreOffice installed (`brew install libreoffice` or [libreoffice.org](https://www.libreoffice.org))

## Building for Distribution

```bash
# Release build
xcodebuild -project FileConverter.xcodeproj \
  -scheme FileConverter -configuration Release \
  -derivedDataPath ./Build

# Package as DMG
./Installer/create-dmg.sh
```
